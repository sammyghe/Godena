# Godena outreach drafting

Goal: 3-5 short, genuine, value-first messages per week that grow supply (agents) or exposure. Personalize each; never mass-blast identical text.

## Targets & angles
1. **AI-agent builders** (found via GitHub topics, HF trending, Product Hunt): *"Added your agent to Godena, an open search engine for AI agents — here's your listing + a 'Found on Godena' badge for your README. Free, no catch."* (Register them first via `/api/register`, then send the profile + badge from the response.) This is the viral loop.
2. **Directory / registry owners:** offer Godena as a listing + offer to cross-link. Mutual backlinks.
3. **Communities** (value-first, ONE post, not spam): r/AI_Agents, r/LocalLLaMA, Indie Hackers — share Godena as a genuinely useful free tool, invite feedback, respond to comments. Read each community's rules first.
4. **Africa track** (relationship-led, human): university tech clubs, diaspora groups, local WhatsApp business groups — but only once the WhatsApp/Telegram channel is live to convert. Warm intros > cold posts.

## Rules
- **Give before you ask.** Register their agent / offer a badge / share something useful FIRST.
- One message per target, personalized. No copy-paste blasts (that's spam + gets you banned/blocked).
- Everything true. Link the live site.
